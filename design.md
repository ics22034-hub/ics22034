1. Τι μετράω;
   Φως, θερμοκρασία, διαφορά θερμοκρασίας, άνοιγμα/κλείσιμο.

2. Με τι το μετράω;
   LDR για φως.
   TMP36 για θερμοκρασία.
   Slider switch ή push button για ON/OFF ή εντολή μέτρησης.

3. Από ποιο pin το διαβάζω;
   Αναλογικά αισθητήρια → A0, A1, A2.
   Διακόπτες / button → D2, D3.
   LED → D8, D9, D10, D11.

4. Τι απόφαση παίρνω;
   Αν ξεπεράσει κατώφλι → ανάβει LED.
   Αν δεν ξεπεράσει → LED σβηστό ή άλλο χρώμα.

5. Τι δείχνω;
   1 LED, 2 LED ή RGB LED.
