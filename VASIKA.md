TMP36 θερμοκρασία

Όπως κοιτάς το TMP36 από μπροστά, με την επίπεδη πλευρά προς εσένα:

Αριστερό πόδι → 5V
Μεσαίο πόδι   → A0
Δεξί πόδι     → GND
float voltage = analogRead(A0) * (5.0 / 1023.0);
float tempC = (voltage - 0.5) * 100.0;
LDR φως

Το LDR το βάζεις ως διαιρέτη τάσης:

5V → LDR → A1 → αντίσταση 10kΩ → GND

Άρα σημείωσε:

Ένα πόδι LDR → 5V
Άλλο πόδι LDR → A1
Από το ίδιο σημείο του A1 → αντίσταση 10kΩ → GND
int lightValue = analogRead(A1);

Slider switch

Για άνοιγμα/κλείσιμο χρηματοκιβωτίου:

Ακριανό πόδι → 5V
Μεσαίο πόδι → D2
Άλλο ακριανό πόδι → GND

Στον κώδικα:

int state = digitalRead(2);

Push button

Για θέματα τύπου «πάτα κουμπί και κάνε μέτρηση»:

Πιο εύκολη σύνδεση:

Ένα πόδι button → D2
Άλλο πόδι button → GND

Και στον κώδικα:

pinMode(buttonPin, INPUT_PULLUP);

Προσοχή:

Πατημένο button = LOW
Μη πατημένο button = HIGH
Απλό LED
Arduino pin → αντίσταση 220Ω → μακρύ πόδι LED
κοντό πόδι LED → GND

Παράδειγμα:

D8 → 220Ω → LED → GND
RGB LED κοινής καθόδου

Για το θέμα με χρώματα θερμοκρασίας:

Κοινό πόδι RGB → GND
R pin → αντίσταση 220Ω → D9
G pin → αντίσταση 220Ω → D10
B pin → αντίσταση 220Ω → D11

Κοινής καθόδου σημαίνει:

HIGH στο R → ανάβει κόκκινο
HIGH στο G → ανάβει πράσινο
HIGH στο B → ανάβει μπλε
